The Data Engineering Zoomcamp is a comprehensive program designed to teach data engineering skills. Throughout the course, I will be covering various modules, including:

-   Docker
-   SQL
-   Terraform
-   Data Warehousing
-   Data Pipelines
-   Analytics Engineering
-   Batch Processing
-   Stream Processing
-   Data Orchestration
